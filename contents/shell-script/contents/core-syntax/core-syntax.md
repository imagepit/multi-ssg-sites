## 📚 はじめに

Shell Scriptのコア構文は、自動化やシステム管理タスクを効率的に実行するための**基礎的な構成要素**です。これらの概念を理解することで、より複雑で実用的なスクリプトを構築できるようになります。

本ページでは、Shell Scriptプログラミングにおいて必須となる6つのコア概念を体系的に学びます。

### 🎯 このページで学べること

:::note

このコア構文ガイドでは、以下の6つの基本概念を体系的に学習できます：

- **変数**: データの格納と管理方法
- **クォート**: 文字列の適切な扱い方
- **条件分岐**: プログラムの流れを制御する方法
- **ループ**: 繰り返し処理の実装
- **関数**: コードの再利用とモジュール化
- **配列**: 複数データの効率的な管理

:::

## 💡 なぜコア構文が重要なのか

Shell Scriptのコア構文は、**システム管理者や開発者にとって必須のスキル**です。これらの基本概念を理解することで：

- **自動化タスクの効率化**: 日々の定型作業をスクリプトで自動化
- **システム管理の簡素化**: 複雑な管理作業を簡潔に記述
- **パイプライン処理の最適化**: 複数のコマンドを組み合わせた高度な処理
- **エラーハンドリングの実装**: ロバストなスクリプトの作成
- **コードの再利用**: 関数化によるメンテナンス性の向上

## 📖 コア構文の全体像

以下の6つの概念がShell Scriptプログラミングの基礎を形成します：

### 🔤 変数 (Variables)

変数は、データを一時的に格納し、後で参照するための基本的な仕組みです。Shell Scriptでは変数を使ってテキスト、数値、パスなどの情報を管理します。

#### 主な学習内容
- 変数の宣言と代入
- 変数の参照と展開
- 環境変数とローカル変数
- 変数のスコープとライフサイクル

#### 具体的な使用例
```bash
#!/bin/bash

# 変数の宣言と代入
name="Taro Yamada"
age=25
home_dir="/home/user"

# 変数の参照
echo "名前: $name"
echo "年齢: $age歳"
echo "ホームディレクトリ: $home_dir"

# 変数を使ったパス操作
config_file="$home_dir/.config/app.conf"
echo "設定ファイル: $config_file"
```

**詳細を学ぶ**: [変数の詳細ガイド](./variables.md)

### 🔤 クォート (Quoting)

クォートは、Shellが文字列をどのように解釈するかを制御する重要な機能です。適切なクォートの使用は、予期せぬ動作を防ぎ、安全なスクリプト作成に不可欠です。

#### 主な学習内容
- シングルクォート (`'`)
- ダブルクォート (`"`)
- バックスラッシュ (`\`) によるエスケープ
- クォートの使い分けとベストプラクティス

#### 具体的な使用例
```bash
#!/bin/bash

# シングルクォート：文字列をそのまま扱う
message='Hello, $name!'
echo $message  # 出力: Hello, $name!

# ダブルクォート：変数展開を行う
name="Taro"
message="Hello, $name!"
echo $message  # 出力: Hello, Taro!

# エスケープ文字
path="C:\\Program Files\\App"
echo "パス: $path"
```

**詳細を学ぶ**: [クォートの詳細ガイド](./quoting.md)

### 🔀 条件分岐 (Conditional Branching)

条件分岐は、プログラムの実行フローを制御するための基本的な構造です。特定の条件に基づいて異なる処理を実行できます。

#### 主な学習内容
- `if` 文による基本的な条件分岐
- `case` 文による多方向分岐
- 条件式の評価方法
- ネストされた条件分岐

#### 具体的な使用例
```bash
#!/bin/bash

# 基本的なif文
age=20
if [ $age -ge 18 ]; then
    echo "成人です"
else
    echo "未成年です"
fi

# case文による分岐
fruit="apple"
case $fruit in
    "apple")
        echo "リンゴです"
        ;;
    "orange")
        echo "オレンジです"
        ;;
    *)
        echo "その他の果物です"
        ;;
esac
```

**詳細を学ぶ**: [条件分岐の詳細ガイド](./conditional-branching.md)

### 🔄 ループ (Loops)

ループは、同じ処理を繰り返し実行するための構造です。ファイル処理、データ変換、バッチ処理などで頻繁に使用されます。

#### 主な学習内容
- `for` ループによる繰り返し処理
- `while` ループによる条件付き繰り返し
- `until` ループによる終了条件指定
- ループの制御（`break`, `continue`）

#### 具体的な使用例
```bash
#!/bin/bash

# forループによるファイル処理
for file in *.txt; do
    echo "処理中: $file"
    # ファイルに対する処理
done

# whileループによるカウントアップ
count=1
while [ $count -le 5 ]; do
    echo "カウント: $count"
    ((count++))
done

# C言語スタイルのforループ
for ((i=0; i<10; i++)); do
    echo "インデックス: $i"
done
```

**詳細を学ぶ**: [ループの詳細ガイド](./loops.md)

### 🏗️ 関数 (Functions)

関数は、コードを再利用可能な単位にまとめるための仕組みです。関数を使うことで、コードの可読性と保守性が向上します。

#### 主な学習内容
- 関数の定義と呼び出し
- 引数と戻り値の扱い
- 関数のスコープ
- 再帰関数の実装

#### 具体的な使用例
```bash
#!/bin/bash

# 関数の定義
greet() {
    local name="$1"
    echo "こんにちは、$nameさん！"
}

# 関数の呼び出し
greet "Taro"

# 戻り値を持つ関数
add_numbers() {
    local num1=$1
    local num2=$2
    echo $((num1 + num2))
}

result=$(add_numbers 10 20)
echo "計算結果: $result"
```

**詳細を学ぶ**: [関数の詳細ガイド](./functions.md)

### 📊 配列 (Arrays)

配列は、複数の値を一つの変数で管理するためのデータ構造です。リストデータの処理や、複数のファイル名の管理などで活用されます。

#### 主な学習内容
- 配列の宣言と初期化
- 配列要素へのアクセス
- 配列の操作（追加、削除、ソート）
- 連想配列の使用

#### 具体的な使用例
```bash
#!/bin/bash

# 配列の宣言と初期化
fruits=("apple" "orange" "banana" "grape")

# 配列要素へのアクセス
echo "最初の果物: ${fruits[0]}"
echo "すべての果物: ${fruits[@]}"

# 配列のループ処理
for fruit in "${fruits[@]}"; do
    echo "果物: $fruit"
done

# 配列への要素追加
fruits+=("mango")
echo "追加後の配列: ${fruits[@]}"

# 連想配列（Bash 4.0+）
declare -A user_info
user_info["name"]="Taro"
user_info["age"]=25
echo "ユーザー情報: ${user_info[@]}"
```

**詳細を学ぶ**: [配列の詳細ガイド](./arrays.md)
