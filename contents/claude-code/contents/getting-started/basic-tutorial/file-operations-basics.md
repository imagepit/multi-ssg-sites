## ファイル操作の基本

新しいファイルを作成します。


2. ファイルの編集

既存のファイルを編集します。

_コマンド実行_

3. ファイルの確認

作成されたファイルを確認します。

_コマンド実行_

## トラブルシューティング

### 一般的な問題

:::note よくある問題と解決策

**Q: ファイルが見つかりません**
A: カレントディレクトリとファイルパスを確認してください。`pwd`と`ls`で現在の場所を確認できます。

**Q: パーミッションエラーが発生します**
A: ファイルの権限を確認してください。`chmod 644 filename`で権限を変更できます。

**Q: ファイルがロックされています**
A: 使用中のプログラムを終了するか、`lsof filename`でロックしているプロセスを確認してください。

**Q: ディスク容量が不足しています**
A: `df -h`でディスク容量を確認し、不要なファイルを削除してください。

:::
