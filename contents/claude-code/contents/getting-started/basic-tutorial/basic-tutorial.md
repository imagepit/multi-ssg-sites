# 基本操作チュートリアル

このチュートリアルでは、Claude Codeの基本操作を段階的に学習します。インストールと認証設定が完了した方を対象に、日常的な開発業務でClaude Codeを活用できる基礎スキルを身につけます。

:::note このチュートリアルで学べること

- Claude Codeの基本コマンドと使い方
- ファイル操作とコード生成の基本
- プロジェクト管理の基礎
- デバッグと問題解決の方法
- 効率的な開発ワークフローの構築

:::

## 前提条件

このチュートリアルを始める前に、以下の準備が完了している必要があります。

- Claude Codeのインストール完了
- 認証設定の完了
- 基本的なコマンドライン操作の理解
- テキストエディタの利用可能
- インターネット接続

## チュートリアルの進め方

このチュートリアルは以下のセクションで構成されています。

1. **コマンドの基礎** - 基本的なコマンド操作を学習
2. **ファイル操作** - ファイルの作成・編集・管理
3. **コード生成** - AIによるコード生成の活用
4. **プロジェクト管理** - プロジェクト全体の管理
5. **デバッグ** - 問題解決とデバッグ技術
6. **実践演習** - 総合的な実践演習

## セクション1: コマンドの基礎

### 基本的なコマンド構造

Claude Codeのコマンドは、以下の基本構造を持っています。

```bash
claude [オプション] [プロンプト]
```

### ヘルプの表示

:::step

1. ヘルプコマンドの実行

まずはヘルプを表示して、利用可能なコマンドを確認します。

_コマンド実行_
```bash
claude --help
```

2. コマンド一覧の確認

表示されたヘルプを確認し、主要なコマンドを把握します。

```bash
# ヘルプ出力例
Claude Code v1.0.0
Usage: claude [command] [options]

Commands:
  init          Initialize Claude Code configuration
  config        Manage configuration settings
  chat          Start an interactive chat session
  generate      Generate code from prompt
  debug         Debug your code
  help          Show help information
```

:::

### バージョン確認

:::step

1. バージョン確認

インストールされているClaude Codeのバージョンを確認します。

_コマンド実行_
```bash
claude --version
```

2. 設定確認

現在の設定状況を確認します。

_コマンド実行_
```bash
claude config list
```

:::

### 最初の対話

:::step

1. 簡単な対話を試す

Claude Codeとの最初の対話を試します。

_コマンド実行_
```bash
claude "こんにちは、Claude Codeの使い方を教えてください"
```

2. コード生成を試す

簡単なコード生成を試します。

_コマンド実行_
```bash
claude "Hello Worldを表示するPythonコードを生成してください"
```

:::

## トラブルシューティング

### 一般的な問題

:::note よくある問題と解決策

**Q: コマンドが見つかりません**
A: PATHが正しく設定されているか確認してください。`which claude`で場所を確認できます。

**Q: 認証エラーが発生します**
A: APIキーが正しく設定されているか確認してください。`claude config list`で設定を確認できます。

**Q: 生成されたコードが動作しません**
A: エラーメッセージを確認し、`claude debug "エラーメッセージ"`で解決策を求めてください。

:::

- キャッシュ機能を活用する
