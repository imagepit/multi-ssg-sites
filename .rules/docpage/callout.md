## コールアウトの書き方

### 専門用語の説明

専門用語やその文章の重要な部分を強調するときには次のように記述します。

```markdown
:::note [専門用語]とは

[必要にあわせて専門用語などの説明を行う]

:::
```

### 構文の説明

構文を説明するときには次のように記述します。

````markdown
:::syntax [構文のタイトル]

```{lang}
[構文を示すコードを記述する]
```

[構文の説明を記述する]

:::
````

### 警告の説明

警告を説明するときには次のように記述します。

```markdown
:::warning [警告のタイトル]

[警告の説明を記述する]

:::
```
